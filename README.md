# PunkApp

Integrated Punk Api https://punkapi.com/documentation/v2 into an Android App. 

      
      Tech Stack Used In the project is 
      1) JetPack Compose
      2) Hilt 
      3) Retrofit
      4) OkHttp
      5) Coil
      6) Junit
      7) MockK
